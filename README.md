# JAVA_LAB

A long-term workspace for mastering the Java language through continuous learning, practice, and exploration.

This repository serves as a permanent, living knowledge base. It is dedicated strictly to the Java programming language—from core syntax and memory management to advanced platform features and modern version updates. It acts as both an active testing ground and an evolving reference guide for pure Java development.

---

## Objectives
- Maintain a deep, evolvable understanding of Java mechanics and the JVM.
- Master modern Java syntax, records, pattern matching, and updates (Java 21/25+).
- Implement efficient Data Structures and Algorithms using native Java.
- Explore advanced paradigms, including the Collections Framework, Streams, and Multithreading.
- Retain a structured codebase for rapid syntax revision and technical interview preparation.

---

## Repository Structure

The codebase is organized conceptually to separate core fundamentals from advanced platform features.

* **`src/main/java/com/codewithmahad/`**
  * `core/` - Syntax, data types, control flow, and memory management.
  * `oop/` - Object-Oriented principles, interfaces, and abstract classes.
  * `collections/` - Native data structures, custom sorting, and Maps.
  * `functional/` - Lambdas, Streams API, and functional interfaces.
  * `concurrency/` - Threads, synchronization, and the concurrency utilities.
  * `modern/` - Experimental and newer features from recent JDK releases.
